How It Works

  1. In FiftyOne, select an image and run the Edit Label in Tool operator (press backtick `)
  2. Copy the URL from the "Open Editor" field in the operator result
  3. Paste it into the text box above and click Open Editor
  4. The label editor will open automatically using your current IP address!
